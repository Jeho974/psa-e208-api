# psa-e208-api

API backend pour Peugeot e208 via psa-car-controller, hébergée sur Render.

Ce repo contient la configuration Docker pour déployer [psa-car-controller](https://github.com/flobz/psa_car_controller) sur Render.com.

## Déploiement

1. Connectez ce repo à Render.com (Web Service → Docker)
2. Ajoutez un disque persistant monté sur `/config`
3. Déployez

Voir le tutoriel complet dans votre projet e208 Tracker.
